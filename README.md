FITNESS TRACKER SYSTEM

# TECHNOLOGIES
HTML,CSS,JS,JSP
SERVLET
JDBC,MYSQL

# DESCRIPTION
The Fitness Tracker System is a web-based application developed using Java, JSP, Servlets, JDBC, HTML, CSS, JavaScript, and MySQL. It helps users track workouts, calories, and fitness progress efficiently through secure login, activity management, dashboard monitoring, and database integration.
